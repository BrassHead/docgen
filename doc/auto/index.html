Project docgen

This program extracts internal program documentation by reading specially formatted comments in the source code. For a description of these comments, see the grammar document.

Usage:
docgen <output directory> <file> [<file>...]
<output directory>
docgen creates html files in this directory.
<file>
input file name (eg: *.h *.cpp *.cc)

The output directory will be filled with:

index.html
class list and definition of global routines and variables
<class>.htm
routine descriptions for each class encountered.
