DocClass

Represents a class. Holds the document attributes of the class. Owns DocItems for each Variable and Function of the class.

DocClass member functions

getFileName() Returns the output filename to use.
getFunction() Returns a pointer to the Function object for the given name.
getVariable() Returns a pointer to the Variable object for the given name.
operator<<() Output the body of a class documentation file.

DocClass::getFileName()

String DocClass::getFileName()

Returns the output filename to use. This doesn't include a directory.
